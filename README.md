# Global Industry Daily Briefing (GIDB)

A professional web application for tracking global industry trends in Robotics, Automotive, Battery, Wiring Harness, and Automation.

## Features
- **Daily Industry Briefing:** Get the latest curated news and major issues in 5 key industrial sectors.
- **Historical Archive:** Every daily briefing is automatically saved and can be accessed by date.
- **Modern Industrial UI:** High-performance, responsive design built with Web Components and modern CSS (OKLCH).
- **Data Persistence:** Uses LocalStorage for seamless tracking without a backend.

## Tech Stack
- HTML5, CSS3 (Modern Features)
- JavaScript (ES Modules, Web Components)
- Lucide Icons
- LocalStorage API
